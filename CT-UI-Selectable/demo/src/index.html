初始化


  import SelectableFactory from '@ct/ui-selectable';
  import '@ct/ui-selectable/lib/selectable.less';

  SelectableFactory(document.getElementById('container'), {
    moveInclude: function (els) {

    },
    moveExclude: function (els) {

    },
    upInclude: function (els) {
      console.log(els.length);
    },
    rangeClasses: ['active'],
  });
名称 说明
{HtmlElement} - el 父容器元素
{object} - config

  • 指示器配置 {object} - config

    moveInclude {Function(Array<HtmlElement>)} - 选取到的元素

    moveExclude {Function(Array<HtmlElement>)} - 未选取的元素

    upInclude {Function(Array<HtmlElement>)} - 选取结束后选取的元素

    rangeClasses {Array<String>} - 选取框的样式
