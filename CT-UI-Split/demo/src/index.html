初始化


import SplitFactory from '@ctsj/ui-split/split';
import '@ctsj/ui-split/split.less';

SplitFactory.create(this.dom, {
  direction: 'horizontal',
});
名称 说明
{HtmlElement} - el 父容器元素
{object} - config

  • 指示器配置 {object} - config

    direction {String} - [horizontal(横) | vertical(竖)]

    direction为horizontal只有ct-split-left和ct-split-right
    direction为vertical只有ct-split-top和ct-split-top

    minBlankWidth {Number} - 横向最小留白距离 - 默认30

    minBlankHeight {Number} - 纵向最小留白距离 - 默认30

    onCanMove {Function} - 可以进行拖动

    onMove {Function} - 拖动中

    onSuccess {Function} - 拖动结束
