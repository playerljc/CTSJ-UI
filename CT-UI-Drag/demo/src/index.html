初始化


import DragFactory from '@ctmobile/ui-drag';
import '@ctmobile/ui-drag/drag.less';
DragFactory.create(document.getElementById('ct-drag'), {
  mode: 'clone',
  cloneClasses: ['cloneNode'],
  showMap: true,
});
名称 说明
{HtmlElement} - el 父容器元素
{object} - config

  • 指示器配置 {object} - config

    mode {String} - [normal(正常模式) | clone(克隆模式)]

    cloneClasses {Array<String>} - mode为clone时移动节点的样式

    showMap {Boolean} - 是否显示地图

    moveStep {number} - 移动的步进 默认值1
